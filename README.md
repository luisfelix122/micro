# Visualizador de Microeconomía Interactivo

Este proyecto permite estudiar de forma visual e interactiva los conceptos clave de los mercados competitivos, intervenciones del estado y el monopolio.

## Cómo usar
Simplemente abre el archivo `index.html` en cualquier navegador moderno.

## Características
- **Paso a paso**: Tutorial interactivo que explica cada gráfico.
- **Gráficos SVG Dinámicos**: Curvas de oferta, demanda, ingresos marginales y beneficios.
- **Estética Premium**: Diseño en modo oscuro con efectos de cristal (glassmorphism).
- **Contenido Educativo**: Basado en los conceptos de excedentes, escasez, cuña fiscal y maximización de beneficios.

Desarrollado por Antigravity (Google DeepMind).
